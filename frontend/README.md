# Everytime Clone - Frontend

React를 사용한 에브리타임 클론 프론트엔드입니다.

## 설치 방법

1. 의존성 설치
```bash
npm install
```

2. 개발 서버 실행
```bash
npm start
```

브라우저에서 http://localhost:3000 으로 접속할 수 있습니다.

## 주요 기능

- 회원가입 / 로그인
- 게시판 목록 조회
- 게시글 작성, 조회, 수정, 삭제
- 댓글 작성, 조회
- 익명 게시 기능
- 좋아요 기능

## 기술 스택

- React
- React Router
- Axios
- Context API (상태 관리)

## 프로젝트 구조

```
src/
├── api/          # API 통신 관련
├── context/      # Context API
├── pages/        # 페이지 컴포넌트
└── App.js        # 메인 앱 컴포넌트
```
